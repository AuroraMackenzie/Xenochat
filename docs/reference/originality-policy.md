# Originality Policy

## Mandatory rules
1. No direct copy/paste from reference repositories.
2. No direct translation of foreign-language comments.
3. UI layout and component hierarchy must remain structurally distinct.
4. Asset files must be independently sourced or generated.
5. Similarity checks must be part of CI before merge.

## Required audits
- Code similarity scan report.
- UI structure and screenshot comparison review.
- Asset hash difference report.
- Comment similarity review.
