## Checklist
- Official API permission is documented.
- Terms of service permit the planned usage pattern.
- Data retention policy is mapped to platform obligations.
- User consent requirements are implemented.
- Bot identity and disclosure requirements are implemented.
- Incident response path is documented.
- Token scope is minimal and rotated.
- Export-only mode is selected if no real-time API is available.
